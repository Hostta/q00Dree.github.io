Вопрос 8. Чем отличаются Web server и Application server?

Web server — приложение, принимающий HTTP-запросы от клиентов, обычно веб-браузеров, и выдающий им HTTP-ответы, как правило, вместе с HTML-страницей, изображением, файлом, медиа-потоком или другими данными.


Application server - это более комплексное ПО, иногда имеет встроенный веб-сервер, занимается выполнением расчетов, взаимодействует с хранилищем данных и другими приложениями.
